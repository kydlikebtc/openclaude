## 三、合作模式设计

### 模式 1：免费体验 + 自发分享（成本最低）

- 提供免费 unlimited API access（3 个月）
- 不要求发帖，仅请求 honest feedback
- 如果体验好，KOL 自然会分享
- **适用于：** Tier 1 大号，不接受赞助的开发者

### 模式 2：赞助评测（中等成本）

- 支付 $200-$1,000 per post（根据粉丝量）
- KOL 做真实使用评测，需标注 #sponsored
- 需要 KOL 提前试用 7+ 天
- **适用于：** Tier 1 中等号，YouTuber

### 模式 3：Referral 佣金（长期）

- KOL 获得专属 referral link
- 每个注册用户奖励：$5 registration + 10% API spend（首 3 个月）
- 月度结算
- **适用于：** 所有 Tier，长期合作

### 模式 4：联合内容（高投入）

- 共同制作 "How I Cut My Claude Costs by 75%" 视频/文章
- OpenClade 提供数据、技术支持
- KOL 提供创意和分发渠道
- **适用于：** 头部 KOL，战略合作

## 四、外联流程 SOP

### Step 1: 名单建设（T-14 至 T-10）

1. 在 Twitter/YouTube/Dev.to 搜索关键词：
   - "claude api cost"
   - "anthropic expensive"
   - "bittensor mining"
   - "ai api pricing"
   - "claude api tutorial"
2. 建立 KOL 数据库（Notion/Sheet）：
   - 账号名、平台、粉丝数、互动率、联系方式
   - 最近相关帖子链接
   - 合作意愿评估（1-5）

### Step 2: 预热互动（T-10 至 T-7）

1. 关注目标 KOL
2. 对其相关帖子进行有价值的评论（非纯推广）
3. RT/引用他们的优质内容
4. 建立认知和善意

### Step 3: 发送 DM（T-7 至 T-5）

1. 使用对应 Template（A/B/C）
2. 个性化首段（必须引用他们的具体内容）
3. 控制在 150 字以内（DM 版简化模板）
4. 附带 1 个链接（不要多）

### Step 4: 跟进（T-3）

1. 仅跟进一次，不反复催
2. 提供额外价值（例如 exclusive 数据、early access）
3. 尊重不回复 = 不感兴趣

### Step 5: Launch Day 通知（D-Day）

1. 向已确认合作的 KOL 发 "We're live!" DM
2. 附 PH 链接和推文链接
3. 感谢他们的支持

### Step 6: 后续跟进（D+7）

1. 分享 Launch 数据
2. 邀请长期合作
3. 讨论 Referral 佣金模式

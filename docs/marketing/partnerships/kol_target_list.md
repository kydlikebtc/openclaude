## 四、外联执行 SOP

### Phase A: 预热（Launch T-14 开始）

```
1. 在 Twitter 上 follow 所有目标 KOL
2. 真诚互动 3-5 天（评论、转发、引用推文）
3. 不要一上来就 pitch — 先建立存在感
```

### Phase B: 首次 DM（Launch T-7）

```
1. 使用 influencer_outreach.md 中的 DM 模板
2. 个性化每条消息（引用他们最近的内容）
3. 提供即时价值：免费无限 API 额度
4. 不要要求任何回报 — 先给价值
```

### Phase C: Launch 通知（Launch T-1）

```
1. 给所有已回复的 KOL 发 "明天上线" 预告
2. 附上 PH 页面链接 + 预览内容
3. 请求 Launch Day 转发/upvote 支持
```

### Phase D: 后续合作（Launch D+3）

```
1. 发送 Launch 数据（注册数、PH 排名）
2. 提出长期合作方案（赞助评测、affiliate）
3. 对于高互动的 KOL，讨论 referral 佣金合作
```

## 一、核心收益公式

```
Miner 月净利润 = (日 Emission × 41% × 你的得分占比 × TAO 价格 × 30) - (日 Token 量 × 官方 API 单价 × 30)
```

## 八、推荐收益叠加效应

### 示例：你引入 10 个活跃用户

假设每个用户月消费 $200，共 $2,000/月：

```
你的推荐用户消费占全网比例：$2,000 / $10,000 全网 = 20%

推荐加成 = min(20% × 2 × 30%, 30%) = min(12%, 30%) = 12%

你的最终得分 = 服务分 × (1 + 12%) = 服务分 × 1.12

💰 相当于服务分额外增加 12%
   如果你原本月净利 $3,330，加成后 → $3,730
```

### 引入 20 个活跃用户（消费 $4,000）

```
占比 40% → 加成 = min(24%, 30%) = 24%
月净利增加 $799 → 总计 $4,129/月
```

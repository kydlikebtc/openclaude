## Outreach Operations SOP

### Pre-Send Checklist

- [ ] Personalize `[Name]`, `[specific content]`, and `[publication name]`
- [ ] Verify their email is current (check bio/website/about page)
- [ ] Confirm they've posted content in the last 30 days (active)
- [ ] Check if they've already covered OpenClade or Bittensor
- [ ] Set follow-up reminders in CRM (Day 3 + Day 7)

### Tracking Spreadsheet Fields

| Field | Description |
|-------|-------------|
| Name | Full name |
| Platform | Twitter / YouTube / Newsletter / Blog |
| Followers | Approximate audience size |
| Category | AI Dev / Crypto / Tech Media |
| Email | Contact email |
| Status | Not Contacted / Email 1 Sent / Email 2 Sent / Replied / Interested / Declined / Published |
| Sent Date | Date of first email |
| Response Date | Date of first reply |
| Notes | Personalization notes, content preferences |

### Response Templates

**If they ask for pricing/discount:**

```
Great question. For content creators, we offer:
- Unlimited free API access for testing/review (no time limit)
- $500 in platform credits to give away to your audience
- 5% revenue share on signups from your referral link (optional)

No exclusivity required. You're free to cover competitors too.
```

**If they ask about risks/ToS:**

```
We're transparent about this. Key points:
1. Miners use legitimate Anthropic API keys they purchased
2. Bittensor is a public decentralized network — we don't control miners
3. We have no indication that Anthropic has taken any enforcement action against this model
4. We publish a full risk disclosure on our site
5. Our gradual rollout guide helps developers maintain Anthropic as a fallback

We believe honesty about risks builds more trust than pretending they don't exist.
```

**If they want to do a sponsored review:**

```
We're open to sponsored content if your audience is a strong fit. Typical terms:
- Flat fee: $500–$2,000 depending on audience size and format
- Must include honest assessment (not just praise)
- Must disclose sponsorship per FTC guidelines
- We review for factual accuracy only (not editorial control)

Alternatively, we can do an affiliate/referral model — 10% revenue share for 12 months on signups from your link.

What works better for your format?
```

## Email Infrastructure

**Recommended tools:**
- **Sending:** Personal Gmail/workspace email (better deliverability for 1:1 outreach)
- **Tracking:** Streak CRM (Gmail integration) or Mixmax
- **Scheduling:** Send Tuesday-Thursday, 9-11 AM recipient's timezone
- **Rate limiting:** Max 20 cold emails/day per email address to avoid spam filters

**Do NOT use:**
- Mass email tools (Mailchimp, SendGrid) for KOL outreach
- Generic templates without personalization
- Follow-ups more than 3 times
- Weekend or holiday sends

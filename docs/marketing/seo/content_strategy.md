## 三、SEO 技术清单

### On-Page 优化

- [ ] 每篇文章有唯一 `<title>` 和 `<meta description>`
- [ ] H1 包含主关键词，H2/H3 包含次关键词
- [ ] URL 结构：`/blog/claude-api-pricing-guide-2026`（短、含关键词、无日期）
- [ ] 图片添加 alt text（含关键词变体）
- [ ] 内链策略：每篇文章至少 2 个内链到其他 OpenClade 页面
- [ ] Schema markup：Article 类型 + FAQ Schema（适用于问答型内容）
- [ ] 移动端适配验证
- [ ] Core Web Vitals 达标（LCP < 2.5s, CLS < 0.1）

### 技术 SEO

- [ ] `sitemap.xml` 生成并提交 Google Search Console
- [ ] `robots.txt` 配置正确
- [ ] 404 页面友好提示 + 引导
- [ ] Canonical URL 设置（避免重复内容）
- [ ] Open Graph 标签（社交分享预览）
- [ ] JSON-LD 结构化数据

### Off-Page / 反向链接策略

| 策略 | 动作 | 预期效果 |
|------|------|----------|
| **Guest Post** | 向 Dev.to, Medium, Hashnode 投稿技术教程 | 高质量 dofollow 链接 |
| **GitHub README 链接** | 在 README 中链接到博客和文档 | 品牌权威 |
| **Reddit 参与** | 在 r/MachineLearning, r/ChatGPT 有价值回复中提及 | 流量 + 品牌认知 |
| **HN 帖子** | Show HN + 技术讨论参与 | 高权重反向链接 |
| **比较网站** | 提交到 AlternativeTo, G2, Product Hunt | 反向链接 + 发现渠道 |
| **开发者工具清单** | 提交到 awesome-lists, AI tools directories | 长尾流量 |
| **Bittensor 生态** | Bittensor 官网生态列表 | 精准受众 |

## 四、内容模板

### 标准博客文章结构

```markdown
# [标题 — 含主关键词]

**TL;DR:** [2-3 句总结]

## The Problem [描述痛点]

## The Solution [介绍方案]

## How It Works [技术解释]

## Step-by-Step [操作指南 / 代码示例]

## Results / Benchmarks [数据证明]

## FAQ [结构化 FAQ — 利于 Featured Snippet]

### [关键词问题 1]?
[回答]

### [关键词问题 2]?
[回答]

## Get Started
[CTA — 免费试用链接]
```

### FAQ Schema 模板

每篇文章底部的 FAQ 应同时满足：
1. **用户**: 直接回答常见疑问
2. **Google**: FAQ Schema 获取 Featured Snippet
3. **SEO**: 自然植入长尾关键词
